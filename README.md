# Итоговый проект

### Тема: Веб-сервер

Главное функциональное требование - отдавать по HTTP некий контент.

Уровни развития:
1. Статическое содержимое (просто отдаём странички, которые у нас лежат на диске)
  - поддержать HTTPS
  - покрыть unit-тестами
  - написать документацию по использованию
2. Изменение содержимого аля форум
  - поддержка запросов от пользователя на добавление данных с сохранением в БД
  - сессионность
  - авторизация
